## Testing Libraries Available

Only `flutter_test` (SDK built-in) is available. To write meaningful tests for this codebase, the following would need to be added to `pubspec.yaml`:

```yaml
dev_dependencies:
  flutter_test:
    sdk: flutter
  mockito: ^5.4.0          # or mocktail for null-safe mocking
  build_runner: ^2.4.0     # required for mockito code generation
  shared_preferences: ^2.3.3  # already in dependencies ‚Äî can use fake impl
```

For Supabase mocking, a local mock client or dependency injection refactor would be required, since all services directly call `Supabase.instance.client`.
