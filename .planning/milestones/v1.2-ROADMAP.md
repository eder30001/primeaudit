## Overview

Adicionou o módulo de Checklist independente ao QAudit — desde o banco de dados e gerenciamento de templates até a execução completa com fotos por item. O módulo é totalmente desacoplado do módulo de Auditoria existente.

### Key Decisions

- Módulo Checklist independente do Auditoria — zero alterações nos serviços existentes
- Seeds com UUIDs hardcoded + ON CONFLICT DO NOTHING — idempotência garantida
- RLS Pattern 3: subquery via FK para itens filhos (sem created_by direto na tabela filha)
- Upload de fotos independente do auto-save — falha não bloqueia execução (core value)
- REP-01/02 (PDF) movidos para v2 — complexidade de compute isolate + FileProvider Android
- Tipos `number` e `date` excluídos do denominador do cálculo de conformidade

### Bug Fixes Included (post-Phase 15)

- Correção de carregamento de templates após migração de banco
- Fix de upload de imagens para roles superuser/dev (policy Storage audit-images)
- Migration da coluna corrective_action_id em audit_item_images + policy UPDATE auditor
- Overflow no dropdown de template na sheet de novo checklist

### Technical Debt

- Assinatura digital ausente — finalização sem assinatura é comportamento atual aceito
