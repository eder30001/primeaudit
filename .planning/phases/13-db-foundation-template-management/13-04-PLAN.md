```dart
Future<void> _loadItems() async {
  if (widget.editing == null) return;
  setState(() => _isLoadingItems = true);
  try {
    final items = await _service.getItems(widget.editing!.id);
    if (mounted) {
      setState(() {
        _items.clear();
        for (final item in items) {
          _items.add({
            'ctrl': TextEditingController(text: item.description),
            'item_type': item.itemType,
          });
        }
      });
    }
  } catch (e) {
    if (mounted) _showError('Erro ao carregar itens.');
  } finally {
    if (mounted) setState(() => _isLoadingItems = false);
  }
}
```

**_save() method:**
```dart
Future<void> _save() async {
  if (!_formKey.currentState!.validate()) return;
  if (_items.isEmpty) {
    _showError('Adicione ao menos um item');
    return;
  }
  setState(() => _isSaving = true);
  try {
    final name = _nameCtrl.text.trim();
    final category = _category!;
    final description = _descCtrl.text.trim().isEmpty ? null : _descCtrl.text.trim();
    final itemMaps = _items.map((e) => {
      'description': (e['ctrl'] as TextEditingController).text.trim(),
      'item_type': e['item_type'] as String,
    }).toList();

    if (widget.editing != null) {
      // Edit mode: update metadata + replace items
      await _service.updateTemplate(widget.editing!.id, name: name, category: category, description: description);
      await _service.replaceItems(widget.editing!.id, itemMaps);
    } else {
      // Create mode: create template + create items
      final newTemplate = await _service.createTemplate(name: name, category: category, description: description);
      await _service.createItems(newTemplate.id, itemMaps);
    }

    if (mounted) Navigator.of(context).pop();
  } catch (e) {
    if (mounted) _showError('Erro ao salvar. Verifique os dados e tente novamente.');
  } finally {
    if (mounted) setState(() => _isSaving = false);
  }
}
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Form input → PostgREST | User-supplied name, description, item descriptions pass through Supabase parameterized queries |
| Form → service (create) | Service hardcodes is_padrao=false and created_by=auth.uid() — form cannot override these |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-13-04 | Tampering | SQL injection via template name / item description | accept | PostgREST uses parameterized queries; Supabase Dart client never interpolates user input into raw SQL. No mitigation needed beyond existing SDK behavior. |
| T-13-10 | Tampering | Form passing is_padrao=true via API | mitigate | createTemplate service method hardcodes `'is_padrao': false`. RLS INSERT policy also enforces `is_padrao = false WITH CHECK`. Double guard. |
| T-13-11 | Tampering | Edit of another user's template | mitigate | RLS UPDATE policy: `created_by = auth.uid() AND is_padrao = false`. Even if UI somehow opens edit for another user's template, DB rejects. |
| T-13-12 | Denial of Service | Unbounded items list | accept | No artificial limit set. Items list is bounded by UX (user manually adds each item). DB has no size limit on the list but this is acceptable for v1.2 scope. |
</threat_model>
