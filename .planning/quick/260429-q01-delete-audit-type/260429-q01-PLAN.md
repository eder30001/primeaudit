# Quick Task 260429-q01: Excluir Tipo de Auditoria

## Goal
Adicionar opção "Excluir" no `PopupMenuButton` de cada tipo de auditoria não-global em `AuditTypesScreen`, com diálogo de confirmação e tratamento de erro de FK (tipo vinculado a auditorias existentes).

## Tasks

### Task 1: Adicionar `_confirmDelete` e opção no menu
- **File:** `primeaudit/lib/screens/templates/audit_types_screen.dart`
- **Action:**
  1. Adicionar `PopupMenuItem(value: 'delete', ...)` no `itemBuilder` do `PopupMenuButton`
  2. Adicionar `if (v == 'delete') _confirmDelete(type)` no `onSelected`
  3. Implementar `_confirmDelete(AuditType type)`: diálogo de confirmação + chamada a `_service.deleteType(type.id)` + tratamento de FK error
- **Verify:** `dart analyze` sem erros; `deleteType()` já existe no service (linha 48 de `audit_template_service.dart`)
- **Done:** Menu mostra "Excluir" vermelho, confirmação aparece, tipo é removido da lista após confirmação

## must_haves
- [ ] Opção "Excluir" visível apenas para tipos não-globais e roles com `_canManage`
- [ ] Diálogo de confirmação antes de qualquer delete
- [ ] Erro amigável quando tipo tem auditorias vinculadas (FK constraint)
- [ ] Lista recarrega após exclusão bem-sucedida
